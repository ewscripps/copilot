# GitHub Copilot Setup Guide

> For E.W. Scripps Employees — New & Existing

Follow the steps below to create your GitHub account, install Visual Studio Code, and get GitHub Copilot running.

---

## Step 1 — Create a GitHub Account

1. Go to **github.com/signup**
2. Sign up using your **E.W. Scripps email address**
3. Create a password and choose a username
4. Complete the verification prompt
5. Check your email for a verification code and enter it on GitHub
6. You can skip or complete the onboarding survey — it won't affect setup
7. Send your GitHub username to **Sean Crain** — he'll link your account to the E.W. Scripps enterprise GitHub instance

> **You now have a GitHub account linked to your E.W. Scripps email.**

---

## Step 2 — Install Visual Studio Code

1. Log in to the **E.W. Scripps company portal**
2. Search for **Visual Studio Code**
3. Click to install — the portal will handle the installation automatically
4. Once complete, launch **VS Code**

> **VS Code is installed and ready.**

---

## Step 3 — Install GitHub Copilot Chat

1. Open the **Extensions** panel — click the icon in the left sidebar or press `Ctrl+Shift+X` (Windows) / `Cmd+Shift+X` (Mac)
2. Search for **GitHub Copilot Chat**
3. Click **Install**

> **GitHub Copilot is installed and active** — Copilot Chat now includes all Copilot functionality including inline code suggestions.

---

## Step 4 — Sign In & Authorize

1. VS Code will prompt you to **Sign In to GitHub** — click it (check the bottom-right corner or the **Accounts** menu)
2. Your browser will open — click **Authorize Visual-Studio-Code**
3. When prompted, click **Open Visual Studio Code** to return to the editor
4. Confirm the **Copilot icon** is visible in the bottom status bar of VS Code

> **GitHub Copilot is active and ready to use.**

---
