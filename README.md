# GitHub Copilot Setup Guide
> For E.W. Scripps Employees — New & Existing

Follow the steps below to create your GitHub account, install Visual Studio Code, and get GitHub Copilot running.

## Step 1 — Create a GitHub Account
Go to github.com/signup
Sign up using your E.W. Scripps email address
Create a password and choose a username
Complete the verification prompt
Check your email for a verification code and enter it on GitHub
You can skip or complete the onboarding survey — it won't affect setup
Send your GitHub username to Sean Crain — he'll link your account to the E.W. Scripps enterprise GitHub instance
✅ You now have a GitHub account linked to your E.W. Scripps email.

## Step 2 — Install Visual Studio Code
Log in to the E.W. Scripps company portal
Search for Visual Studio Code
Click to install — the portal will handle the installation automatically
Once complete, launch VS Code
✅ VS Code is installed and ready.

## Step 3 — Install GitHub Copilot Chat
Open the Extensions panel — click the icon in the left sidebar or press Ctrl+Shift+X (Windows) / Cmd+Shift+X (Mac)
Search for GitHub Copilot Chat
Click Install
✅ GitHub Copilot is installed and active — Copilot Chat now includes all Copilot functionality including inline code suggestions.

## Step 4 — Sign In & Authorize
VS Code will prompt you to Sign In to GitHub — click it (check the bottom-right corner or the Accounts menu)
Your browser will open — click Authorize Visual-Studio-Code
When prompted, click Open Visual Studio Code to return to the editor
Confirm the Copilot icon is visible in the bottom status bar of VS Code
✅ GitHub Copilot is active and ready to use.
